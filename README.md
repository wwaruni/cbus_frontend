# Cbus Customer Dashboard Front End

## Prerequisits
* Latest Node version (v25.6.0)
* yarn or npm for package management (yarn version v1.22.22)
  
##  Local Setup

* checkout code repo
* run ` yarn`. This will install all the node modules
* `yarn dev`
* Make sure to have a .env file based on env.sample and replace that with real values

## To run tests 

* `yarn test`
